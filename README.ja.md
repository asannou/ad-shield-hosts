# Ad-Shield Hosts Generator

このリポジトリは、Ad-Shield に関連するドメインをブロックするための `hosts` ファイルを自動生成・更新するツールを含んでいます。

## 概要

メインスクリプトは、難読化された Ad-Shield のスクリプトを解析して配信ドメインを抽出します。抽出されたドメインは、広告やトラッキングをブロックするための標準的な `hosts` ファイル形式（`0.0.0.0 domain`）で出力されます。

## リポジトリの構成

- **`index.js`**: ドメイン抽出のコアロジックです。スクリプトを取得し、難読化を解除し、AST（抽象構文木）を走査して特定のパターンを特定し、見つかったドメインを出力します。
- **`hosts`**: 抽出されたドメインを含む、自動生成されたブロックリストファイルです。
- **`.github/workflows/generate.yml`**: `hosts` ファイルを定期的に更新し、変更を自動的にコミットする GitHub Actions ワークフローです。
- **`package.json`**: プロジェクトの依存関係を定義しています。

## 使い方

ローカルでリストを生成するには：

1. 依存関係をインストールします：
   ```bash
   npm install
   ```

2. スクリプトを実行します：
   ```bash
   node index.js
   ```

## 自動化

このプロジェクトには、以下の処理を行う GitHub Action (`generate.yml`) が含まれています：
- 1時間ごとに実行。
- 生成スクリプトを実行。
- `hosts` ファイルを更新およびソート。
- 変更があればリポジトリにコミットおよびプッシュ。
